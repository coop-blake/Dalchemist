# Connection
## Windows
A connection to the Catapult Database can be made SQL Anywhere  
  
Run (Windows+R)
  * dbisql.exe
  * dbisql.com

Check connection/database from Powershell  
`dbinfo -c "Host=192.168.0.38;UserID=****;Password=****"`  
Replace **** with username and password
